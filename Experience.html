Installer manuellement RECOMMANDATIONS

Voici la balise Google pour ce compte. Copiez-la et collez-la dans le code de chaque page de votre site, directement après l'élément <head>. N'ajoutez qu'une seule balise Google par page.

                
                    <!-- Google tag (gtag.js) -->
                    <script async src="https://www.googletagmanager.com/gtag/js?id=G-RVH70FFF1L"></script>
                    <script>
                      window.dataLayer = window.dataLayer || [];
                      function gtag(){dataLayer.push(arguments);}
                      gtag('js', new Date());

                      gtag('config', 'G-RVH70FFF1L');
                    </script>
